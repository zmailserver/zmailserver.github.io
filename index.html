ZMailServer

Open Source Email and Collaboration Server

About ZMailServer

ZMailServer is an email, calendar and collaboration server built for the enterprise. The project is a fork of Zimbra with a focus on added open source features and open source community friendliness.

Benefiting from years of experience of the original project, ZMailServer is a stable, full-featured enterprise-class email server. ZMailServer introduces several changes listed in our roadmap.

Features

  • Email
  • Addressbook
  • Calendar
  • Antispam, antivirus
  • Graphical administration console

Community

The source code is easily accessible using git on GitHub. Community contributions are welcomed and encouraged.
